# Ecommerce Webpage Demo

**TheSellForBuy** is a minimalist and modern e-commerce demo website designed to showcase front-end development skills, UI/UX design principles, and interactive features. The project demonstrates a clean, responsive interface that is both user-friendly and visually appealing.  

> **An example of a dysfunctional e-commerce website with a great interface, written with just one page of code.**

---

## Features

- **Minimalist Design:** Clean layout and grid-based product gallery for an uncluttered user experience.  
- **Dark/Light Mode:** Toggle between dark and light themes for optimal viewing comfort.  
- **Multi-Currency Support:** Switch between USD, EUR, and TRY to see prices in different currencies (note: prices are for demonstration purposes only).  
- **Multi-Language Support:** The site supports Turkish (TR), English (EN), Spanish (ES), and German (DE). All product descriptions, labels, and modal info update dynamically.  
- **Responsive Gallery:** Adaptive grid layout ensures the site looks good on all screen sizes.  
- **Interactive Modals:** Click a product to see a detailed modal with description, price, brand, seller info, and estimated delivery.  
- **Live Clock:** Real-time clock displayed in the header for added functionality.

## Installation & Usage

1. Clone this repository:
   ```bash
   git clone https://github.com/aozdev/TheSellForBuy.git
````

2. Open `index.html` in your browser.
3. Explore the gallery, toggle themes, switch currencies, and change languages.

> No backend or database is required; this is a fully front-end static demo.
